viterm ロゴ候補

viteflow system のブランドグラデーションとネイビーを継承した5案。各案はアプリアイコン(大/小)とワードマークのロックアップ。

01

Prompt — ❯ とカーソル

ターミナルの原型「❯_」をそのままマークに。グラデーションのプロンプトが viteflow の血筋を示す。最も説明不要で、メニューバーやDockでも一目でターミナルとわかる。

02

Merge-V — ブランチが合流する「v」

gitグラフの「2本のブランチがマージされる」形がそのまま「v」になる。viterm の核心機能(worktree並列作業→マージ)を1つの字形で語る案。コミットドットがアクセント。

アプリアイコン 128px / 32px
ワードマーク(ライト)
03

Split — 分割ペインとフロー

ウィンドウの分割ペインをアイコン化。左ペインはプロンプト(あなた)、右ペインは流れるライン(働くエージェント)。「並列運用」というプロダクトの主張が最も直接的。

アプリアイコン 128px / 32px
ワードマーク(ライト)
04

Fork — プロンプトから枝分かれ

プロンプト「❯」の先端がworktreeのように枝分かれする。ターミナル(❯)と並列ブランチ(fork)の合成で、01と02の中間。動きのあるストロークが viteflow のマークと響き合う。

アプリアイコン 128px / 32px
ワードマーク(ライト)
05

Flow-V — viteflowマークの直系

viteflow system のロゴマーク(流れる連続ストローク)の「v」部分だけを切り出した直系デザイン。会社ブランドとの親子関係が最も強く、プロダクトファミリーとして展開しやすい。
